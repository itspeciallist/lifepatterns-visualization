рЃњрЃљрЃЏрЃЮрЃЏрЃАрЃљрЃ«рЃЋрЃћрЃџрЃЮрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ: рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃЦрЃЏрЃюрЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃа рЃцрЃЮрЃарЃЏрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃЌрЃљ рЃерЃЮрЃарЃўрЃА рЃљрЃарЃўрЃА:

Phyllotaxis Spiral (рЃцрЃўрЃџрЃЮрЃбрЃљрЃЦрЃАрЃўрЃА рЃАрЃърЃўрЃарЃљрЃџрЃў) - рЃЏрЃфрЃћрЃюрЃљрЃарЃћрЃЌрЃљ рЃќрЃарЃЊрЃўрЃА рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў.

Rose Curve (рЃЋрЃљрЃарЃЊрЃўрЃА рЃЎрЃБрЃЏрЃБрЃџрЃў) - рЃърЃЮрЃџрЃљрЃарЃБрЃџрЃў рЃњрЃарЃљрЃцрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЦрЃЏрЃюрЃўрЃА рЃцрЃџрЃЮрЃарЃљрЃџрЃБрЃа, рЃБрЃџрЃљрЃЏрЃљрЃќрЃЮ рЃцрЃЮрЃарЃЏрЃћрЃЉрЃА.

Lissajous Curve (рЃџрЃўрЃАрЃљрЃЪрЃБрЃАрЃўрЃА рЃарЃЎрЃљрЃџрЃў) - рЃърЃћрЃарЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃњрЃљрЃюрЃбрЃЮрЃџрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЦрЃЏрЃюрЃўрЃА рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃа рЃЎрЃЮрЃџрЃўрЃЋрЃўрЃЎрЃБрЃа рЃарЃЎрЃљрЃџрЃА.

Ulam Spiral (рЃБрЃџрЃљрЃЏрЃўрЃА рЃАрЃърЃўрЃарЃљрЃџрЃў) - рЃАрЃърЃўрЃарЃљрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЊрЃљрЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃърЃарЃљрЃўрЃЏрЃБрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ.

рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ: рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃўрЃА рЃЊрЃарЃЮрЃўрЃА (alive seconds) рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃАрЃЌрЃљрЃю рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃарЃўрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
